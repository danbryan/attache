Attaché's companion now reads your agents in real time. Instead of guessing from how long a session has been quiet, it hears the moment a turn finishes, needs you, or gets going, straight from Claude Code, and reacts with a face full of new expressions.

## What's new

- **Real-time status, not guesswork.** Attaché installs a couple of Claude Code hooks so it knows the instant a turn is done, a session needs you, or a new turn starts. No more a "finished" check on a session that is still thinking, and no more "waiting on you" on one that is just running a long command.
- **It stays lively, and needs-you sticks around.** A session that needs you gets a quick alert, then the pet moves on and keeps reacting to everything else. The reminder lives on as a pinned question-mark badge in the ring, so you can leave two or three and get to them in your own time. A celebration plays right over the top.
- **A face for every moment.** It waves hello when a session starts and goodbye when it ends, rolls its eyes while a session is being set up, squishes flatter and wider as it compacts context (with a squeeze symbol overhead), holds up a green and a red flag to ask permission, shakes a red one on a denial, and crosses its eyes on an API error. Sub-agents ripple around it with a live count.
- **Precise agent status is a setting.** The hooks install on by default and sit alongside your own Claude Code hooks, untouched, with a backup of your settings. Turn them off anytime in Settings, "Precise agent status."

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.

More at https://attache.fm
