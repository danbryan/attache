Attaché 0.5.0 is a broad UX, privacy, responsiveness, and agent-support release: faster, quieter, more private, and now watching more of your agents.

## What's new

- **Noticeably faster.** Launch no longer blocks on enumerating system voices, Settings panes stop re-rendering during narration, and expanding Context → Advanced is instant. A new main-thread watchdog surfaces any stall in Settings → About → Responsiveness.
- **A real voice picker.** Choosing a character's voice is now a searchable, filtered list grouped by language (engine, quality, and language filters), modeled on the macOS voice picker instead of one long flat list. Voice settings that used to live app-wide now belong to the character.
- **More agents, watched.** Attaché now watches Grok Build and opencode sessions alongside Claude Code and Codex, with a local-model badge for Ollama-backed sessions. Any harness can also post events to a documented local integration endpoint.
- **Private, visibly.** Start a private call, or make an in-progress call private (it retroactively scrubs what was already saved, and fails closed if it can't). Mark any watched session "don't record", or forget everything recorded from it. Private mode shows a clear PRIVATE badge.
- **Sharper captions.** Long checksums, URLs, and identifiers now wrap and highlight progressively instead of overflowing, with multilingual segmentation for languages that don't use spaces.
- **Right-click everything.** The bottom-bar controls now have context menus, with the macOS Option-key convention for power actions.
- **Keyboard niceties.** Number keys switch personalities in the palette, j/k navigate the inbox and history, and an optional global hotkey summons Attaché from any app.
- **A more careful character.** The character damps its motion while you read captions, holds still under overlays, and honors Reduce Motion. Colt is now the robot in a cowboy hat, mustache, and bandana, with a live animated mouth.
- **Bounded recap.** Recapping a large inbox stages the work and shows a cost preview instead of failing.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
