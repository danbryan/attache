Attaché keeps your AI agents' updates flowing to you out loud. This release makes the app easier to keep current.

## What's new

- The menu-bar dropdown now shows the version at the top, so you can tell at a glance which build you are on.
- Added **Check for Updates…** to the menu-bar dropdown, alongside a proper **About Attaché** panel, so you can pull the latest without hunting through menus.
- Also carries the fix for a crash that could hit the session switcher (⌘K) on a fresh, quarantined install.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.

More at https://attache.fm
