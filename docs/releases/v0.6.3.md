Attaché 0.6.3 fixes three things people feel immediately.

## Fixed

- **Claude subscription works from the app.** Attaché's hardened subprocess environment omitted the user identity variable that Claude Code's credential lookup depends on, so a machine could be logged in at the terminal while Attaché reported it wasn't. Identity basics now pass through (secrets still never do), and any CLI failure surfaces the real error and the exact binary Attaché ran.
- **Focusing a session never narrates the past.** Focusing or watching a session used to speak its last finished turn as a fresh voicemail. Attaché now checkpoints at the transcript's current end when it starts watching, so only what happens after you focus is narrated.
- **Spoken audio is as loud as your music.** Both the premium voice and the system voice played several times quieter than typical media. All speech now lands at the standard speech loudness with a transparent peak limiter, so your system volume is the only volume control. Cached audio re-renders automatically.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
