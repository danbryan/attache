Attaché 0.6.4 makes karaoke captions trustworthy on every voice.

## New

- **A CC button in the player.** Toggle captions like on any TV: filled when on, dim when off, one press, remembered. A caption style choice (karaoke or plain) lives in Voice & Captions.
- **Exact caption timing for every voice.** Attaché now aligns each spoken clip to its transcript on this Mac, so the highlighted word matches what you hear on the premium voice, system voices, and cloud voices alike, not just ElevenLabs. Playback never waits on alignment: captions start conservatively and switch to exact timing the moment it is ready, then replays are exact from the first word.
- **No more misleading bounce.** When timing for a clip is only an estimate, captions render as plain text instead of pretending to know the word, and upgrade to karaoke as soon as the exact timing lands.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
