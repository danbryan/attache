Attaché now lets you build the character you want to spend time with. A character is one complete loadout: appearance, personality, voice, model, reasoning, pace, and fallbacks, all switching together.

## What's new

- **A game-like Character Studio.** Choose Attaché, Colt, or Echo from the wardrobe, name the character, start from a personality preset or write your own, select an explicit voice and model, and preview the result before saving. The studio is a normal movable and resizable macOS window.
- **One character, one complete loadout.** Every character owns its prompt, voice, model, supported reasoning level, playback pace, and ordered fallback providers. Nothing silently follows a separate app-wide voice or model.
- **Reasoning that matches the model.** Attaché asks the selected provider which reasoning levels the model supports and only offers valid choices. Unsupported reasoning controls stay disabled instead of saving a setting the model cannot use.
- **Fast, expressive switching.** The bottom-bar picker now shows character artwork. Switching changes the personality, voice, model, and presence together with a short visual transition, without an unsolicited greeting.
- **Another take.** Replay any voicemail update through a different character for a fresh interpretation in that character's own voice. During a live conversation, you can also ask another character to clarify without changing the frozen agent destination.
- **Echo joins the same world.** The abstract voice-bars character uses the same session orbit, focus behavior, status reactions, flags, celebrations, and activity effects as Attaché and Colt. Double-click Echo to enter or leave real macOS full screen.
- **Safer audio.** Attaché never starts speaking while you are off-call unless you explicitly press Play on a voicemail or Preview in Character Studio. Preview is the only automatic greeting in the creation flow.
- **A cleaner first run.** New users choose or build a character during onboarding, connect a model with revealable API-key fields, and arrive in the app with the voice and model already attached to that character.
- **Simpler local models.** Ollama is the supported local-model path. LM Studio has been removed from the setup experience.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
