Attaché is a native macOS app that watches the AI agents working for you and briefs you out loud, in a voice and personality you choose.

## What it does

- Speaks each agent turn as a short voice memo, in real time.
- Files updates like voicemail: play all, skip, replay, or recap a batch in one pass.
- Interrupts you only when an agent is actually blocked on you.
- Go live by voice: ask what's running and push new direction back to your agents.
- Any voice, and any personality you can describe.
- Watches OpenAI Codex and Claude Code out of the box, and only speaks about the sessions you pin.
- Run it fully local with open models and on-device voices, or reach for the cloud (ElevenLabs, xAI, OpenAI, Claude) on the work that isn't sensitive.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.

More at https://attache.fm
