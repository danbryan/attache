Attaché 0.6.7 lets you clear out History on your terms.

## New

- **Select and delete History items.** History now has the same multi-select as the inbox: check the items you want, then delete them, with a Select all option and Command-Delete support. Because deleting from History is permanent, it asks for confirmation first. The seven-day retention default still handles automatic cleanup; this is for when you want something gone now.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
