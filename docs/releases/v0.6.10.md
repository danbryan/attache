Attaché 0.6.10 gives the robot and Colt a livelier mouth.

## Changed

- **A more expressive mouth.** The robot and Colt now speak with a richer seven-bar mouth, a touch taller and snappier, with a soft brand-colored glow so it reads as lit from within rather than flat. It still tracks the real audio, and the resting face is unchanged.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
