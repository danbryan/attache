# Attaché 0.6.12

Bring your own appearance, and more room to work in.

## New
- Custom appearances. Give your Attaché a custom face that mounts into the same
  living presence the built-in characters use, with full 360-degree eye tracking
  and the shared voice-reactive mouth. Use the new Import button in a
  personality's Appearance section to add an appearance package from disk.
- Two ready-made examples, a panda and a fox, ship in the project to import or
  remix, along with a full guide to drawing your own (including a prompt you can
  hand to an AI agent).

## Changed
- The personality editor and Settings now scale with the window, so a long
  personality prompt and the busier panes have more room and less scrolling.
- "Presence" is now called "Appearance."
- Echo leans toward the session it is focused on.

## Fixed
- The appearance picker no longer pushes the Voice and Model settings off the
  edge of the window when several appearances are listed; the tiles wrap instead.
