Attaché 0.6.0 rebuilds memory around explicit trust, cleans up onboarding, and makes the premium voice and live calls behave the way they always should have.

## What's new

- **Memory that belongs to your Attaché.** Say "remember this" in a call and it saves, first try, to the Attaché you told it to. Each Attaché keeps its own memories; facts every Attaché should know are typed once under Settings → Memory → Applies to all Attachés. Everything is stored only on this Mac, a quiet "Memory saved" chip confirms each save, and every item can be edited, deleted, or marked Local only so it is never sent to a cloud model. Secrets, credentials, and passing moods are refused by a local safety check no matter what.
- **Premium voice, unbroken.** A sampler bug could make Azelma turn metallic at specific words and replay the glitch identically forever; synthesis now integrates more carefully and cached audio re-renders clean. Voice warnings also tell the truth: if a chosen Apple voice isn't downloaded or the premium voice isn't installed, the editor says exactly that and where to fix it, and any fallback voice is disclosed at the moment it plays.
- **Live calls stay live.** Updates from the session your call is focused on are narrated in the call and filed straight to History as heard. No more unread inbox badges for things you already listened to. Watched-but-unfocused sessions still leave voicemails.
- **Cleaner onboarding.** OpenAI joins the connect-a-model choices, consent wording says plainly that you acknowledge once per provider, the redundant Use button is gone (clicking a connected provider applies it), built-in personalities inherit whatever model you connect instead of presuming one, "Create your own Attaché" works, and source detection refreshes live while the sheet is open.
- **A calmer MCP pane.** Servers detected in your other tools now collapse to a single line until you choose to expand them.
- **Consistent previews.** "Preview personality" speaks the same greeting every click (a New take button regenerates it), so you can actually compare voices.

## Fixes

- Resetting data no longer fabricates a phantom "My Attaché" personality from your own onboarding picks.
- Customizing a built-in personality saves cleanly; a personality without a pinned model is a first-class "uses your connected model" state.
- Onboarding source detection no longer overcounts Claude Code sessions or misses archived Codex sessions.
- Saved memories carry real timestamps.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
