Attaché 0.6.5 makes voicemails behave like voicemails and makes the activity ring tell the truth.

## Fixed

- **Played voicemails move to History.** A voicemail used to leave the inbox only if you listened to the very end. Now, once you have heard at least half of it, it moves to History on any exit, finishing, pausing, stopping, or playing the next one, so the inbox stays current and History fills up as expected. A barely-started message stays in the inbox so nothing is lost.
- **Activity labels reflect real tools.** The orbiting "checking Coinbase" style labels used to fire on any mention of a word in an agent's tool activity, so a Slack search that referenced Coinbase could read "checking Coinbase". Labels now come from the actual tool that ran, so a Slack call reads "checking Slack" and a service name appears only when that service's tool really executed. This works for any tool you add, with nothing to configure.

## New

- **Smart activity ranking.** When an agent is doing many different things at once, Attaché can ask your configured model to surface the few most relevant, so the ring stays readable. Only tool names are ever shared, never their arguments or results, and you can turn it off under Activity insights.

## Install

Download `Attache.dmg` below and drag Attaché to Applications, or:

```
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon. Signed with a Developer ID, notarized, and stapled.
